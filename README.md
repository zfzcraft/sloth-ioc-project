# Sloth-IOC
Sloth IOC —— 云原生极致懒加载 IOC 容器 专为 Kubernetes、微服务、Serverless 设计的革命性极简 IOC 容器 基于 Pure IOC 深度进化，彻底颠覆传统 IOC 架构，启动速度与业务规模完全无关 核心特性 彻底剔除 BeanDefinition，架构极简无冗余 编译时索引 + Maven 插件，运行时零扫包、零类加载、零预热 启动耗时仅由 @Configuration 与插件数量决定，恒定 O (1) 启动开销 极致懒加载：不使用 → 不加载类、不初始化对象 兼容全生态扩展、插件、MVC、AOP、事务能力 百万 Bean 规模依旧 300~500ms 稳定启动 超低内存、无资源损耗，原生适配云原生弹性扩缩容
